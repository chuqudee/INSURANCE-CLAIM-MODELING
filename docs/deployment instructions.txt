## Deployment Instructions

Firstly, the user should have installed the following packages.

- Python
- Streamlit
- Xgboost
- Scikitlearn

Run the app on your command file by typing:
streamlit run claim_prediction.py
then press enter.

your default browser will immediately open the  app page.

if it doesnt, copy the link displayed on the command prompt.

while the app runs, provide the neccesary inputs and click on the predict button.
